## GitHub Pages setup

1. Upload all files inside this project folder to the GitHub repo root.
2. Make sure the `assets` folder uploads with its files inside it.
3. In GitHub go to **Settings → Pages**.
4. Source: **Deploy from branch**.
5. Branch: **main**.
6. Folder: **/ (root)**.
7. Set custom domain to `accountants.kronatrix.co.uk`.
8. Enable HTTPS when available.
